# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased] - 2026-01-16

### 新增功能 (Added)
- **AI 圆桌会议流程**: 实现了包含“开场介绍”、“轮流发言 (Statements)”和“自由辩论 (Debate)”三个阶段的会议状态机。
- **主持人角色 (Host)**: 引入了专门的“主持人”角色来掌控会议节奏、介绍话题并分配发言权。
- **参与者列表栏**: 在聊天室顶部增加了可横向滚动的参与者列表，方便快速查看和互动。
- **快速提及功能 (@Mention)**: 点击参与者头像可自动输入 `@角色名`，并强制指定该 AI 角色优先回答下一个问题。
- **会议阶段指示器**: 在聊天标题栏增加了可视化的阶段标签（Intro, Statements, Debate），实时展示会议进程。

### 优化改进 (Changed)
- **聊天 UI 升级**: 重构了 `ChatBubble` 组件，将角色名称从气泡内移至气泡上方独立显示，提升了阅读体验和界面整洁度。
- **系统消息优化**: 用生动的主持人开场白替换了原本单调的系统“开始”提示。
- **AI 上下文增强**: 优化了 Prompt 工程，让 AI 角色能够感知当前所处的会议阶段及其角色任务（如：在轮流发言阶段简短陈述，在辩论阶段积极互动）。

### 问题修复 (Fixed)
- **DeepSeek/推理模型适配**: 增加了正则过滤逻辑，自动移除 AI 响应中可能包含的 `<think>` 标签，确保输出纯净内容。
- **OpenAI 连接修复**: 修复了 API Endpoint 拼接时可能出现的双重 `/v1` 后缀问题，解决了连接失败的错误。
- **消息去重**: 修复了 React Strict Mode 在初始化时导致主持人开场白重复显示的问题。
- **翻译缺失**: 补充了主持人角色名称及开场白在语言文件中的缺失翻译。
